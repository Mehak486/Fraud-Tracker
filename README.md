# Unified Fraud Tracker

A machine learning based fraud detection system built using Python and Flask
to identify fraudulent and non-fraudulent transactions.

## Features
· Upload transaction dataset (CSV)
· Detect fraudulent transactions using ML models
· Web-based dashboard using Flask
· Data visualization for fraud analysis
· Real-time prediction support

## Technologies Used
· Python
· Machine Learning
· Flask
· HTML, CSS, JavaScript
· SQL / CSV Dataset
· Matplotlib / Seaborn

## How to Run
1. Install dependencies
   pip install -r requirements.txt
2. Run application
   python app.py
3. Open browser
   http://127.0.0.1:5000/
